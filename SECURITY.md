# Security Policy

## Supported Versions

We support security fixes for the latest release on the main branch.

## Reporting a Vulnerability

Please report security issues privately using GitHub Security Advisories:

https://github.com/goosewin/gralph/security/advisories/new

Please avoid opening public issues for security reports.

When reporting, include the gralph version (`gralph version`), OS/arch, and
any backend CLI versions involved.
